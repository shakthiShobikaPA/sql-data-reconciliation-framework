# sql-data-reconciliation-framework
## Business Problem

Data moving between systems can result in missing records, extra records, duplicate records, and field mismatches.

This project compares source and target datasets using SQL validation techniques and calculates data accuracy metrics.

## Tools Used

- SQL
- Excel
- Data Analysis

## Objectives

- Identify Missing Records
- Identify Extra Records
- Detect Field Mismatches
- Measure Data Accuracy
- Generate Reconciliation Reports

## Skills Demonstrated

- SQL
- Data Validation
- Data Quality Analysis
- Data Reconciliation
- Root Cause Analysis
